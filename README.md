#### Quick set-up guide
Go to the directory you want to set this project up in. Then run:

```
git clone https://github.com/alyssa9koh/react-simple-blog.git
npm install
npm start
```

You should be able to see the app at [http://localhost:3000/](http://localhost:3000/).
